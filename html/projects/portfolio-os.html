Portfolio OS

Type Web-based Desktop Environment
Tagline A playful OS inside my portfolio

Portfolio OS is a fully functional desktop environment that runs entirely in your web browser. It includes a complete window manager where you can open, close, minimize, maximize, drag, and resize windows just like a real operating system. The dock contains applications that can be launched, and there is a settings panel for preferences and a notification system. Canvas-based applications include a Tamagotchi pet that roams around the desktop and a CHIP-8 emulator capable of running retro games. All window positions, preferences, and application state are saved using localStorage, so your setup persists between visits. The system handles window focus, z-index management, and window lifecycle events to create an authentic desktop experience without requiring any server-side components or external dependencies.
